# 📖 Overview

**Unemployment Analysis with Python** is a data-driven project focused on analyzing unemployment trends in India.

The project uses Python and popular data science libraries to transform raw unemployment data into meaningful insights.

The analysis focuses on:

* 📈 Unemployment trends over time
* 🦠 Impact of COVID-19 on unemployment
* 🗺️ Region-wise unemployment comparison
* 🏙️ Rural vs Urban unemployment
* 📅 Monthly and seasonal patterns
* 📊 Employment and labour participation analysis
* 🤖 Machine Learning-based unemployment prediction

# 📊 Dataset Information

The project uses the **Unemployment in India** dataset.

The dataset contains information related to:

| Feature                     | Description                                                            |
| --------------------------- | ---------------------------------------------------------------------- |
| Region                      | Indian state or region                                                 |
| Date                        | Date of observation                                                    |
| Frequency                   | Data frequency                                                         |
| Estimated Unemployment Rate | Percentage of unemployed people                                        |
| Estimated Employed          | Estimated number of employed people                                    |
| Labour Participation Rate   | Percentage of working-age population participating in the labour force |
| Area                        | Rural or Urban                                                         |

# 🦠 COVID-19 Impact Analysis

The project analyzes the impact of COVID-19 on unemployment rates.

### 📉 Before COVID-19

Unemployment remained relatively stable before the pandemic.

### 🚨 During COVID-19

Unemployment increased significantly due to:

* 🔒 Nationwide lockdowns
* 🏢 Business closures
* 🏭 Factory shutdowns
* 🚫 Travel restrictions
* 👷 Loss of informal employment
* 📉 Reduced economic activity

The analysis identifies **April 2020** as one of the most severely affected periods.
